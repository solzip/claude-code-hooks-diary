# 📓 Claude Code Working Diary

> Claude Code에서 작업한 내용을 **자동으로** 일일/주간 작업일지로 기록하는 시스템
>
> Automatically logs your Claude Code sessions into daily/weekly work diaries

## 어떻게 동작하나요? / How it works

```
Claude Code 세션 종료 / Session ends
        │
        ▼
  Stop Hook 실행 ──→ working-diary.py
        │
        ▼
  transcript.jsonl 파싱
  (사용자 요청, 수정 파일, 실행 명령어 추출)
        │
        ▼
  ~/working-diary/2026-03-17.md 에 자동 기록
```

## 지원 환경 / Supported Platforms

| Platform | Python | 자동 일지 | 주간 요약 | Cron |
|----------|--------|-----------|-----------|------|
| macOS | python3 | ✅ | ✅ | ✅ |
| Linux | python3 | ✅ | ✅ | ✅ |
| Windows (Git Bash) | python | ✅ | ✅ | ❌ (Task Scheduler 사용) |

## 기록되는 내용 / What gets logged

| 항목 / Item | 설명 / Description |
|------|------|
| 📋 작업 요청 / Task Requests | 사용자가 Claude에게 요청한 내용 |
| 📄 생성된 파일 / Files Created | 새로 만들어진 파일 목록 |
| ✏️ 수정된 파일 / Files Modified | 편집된 파일 목록 |
| ⚡ 주요 명령어 / Key Commands | 실행된 중요 shell 명령어 |
| 📝 작업 요약 / Work Summary | AI가 수행한 작업의 요약 |
| ⚠️ 이슈 / Issues | 발생한 오류나 문제 |

## 설치 / Installation

```bash
# 1. 다운로드
git clone <repo> ~/working-diary-system
# 또는 tar.gz 압축 해제

# 2. 설치 스크립트 실행
cd ~/working-diary-system   # 또는 working-diary-system/
./install.sh                # Windows Git Bash에서도 동작
```

설치 스크립트가 자동으로:
- `~/.claude/hooks/` 에 스크립트 복사
- `~/.claude/settings.json` 에 Stop Hook 등록
- `~/working-diary/` 디렉토리 생성
- Python 명령어 자동 감지 (`python3` 또는 `python`)
- (Unix only) 주간 요약 cron job 등록

## 디렉토리 구조

```
~/working-diary/
├── 2026-03-15.md          ← 일일 작업일지 / Daily diary
├── 2026-03-16.md
├── 2026-03-17.md
├── .session_counts.json    ← 내부 카운트 (자동)
├── .gitignore
└── weekly/
    ├── W11_2026-03-09.md   ← 주간 요약 리포트 / Weekly report
    └── W12_2026-03-16.md
```

## 환경변수 설정 / Configuration

| 환경변수 | 설명 | 기본값 |
|----------|------|--------|
| `CLAUDE_DIARY_LANG` | 언어 / Language (`ko` or `en`) | `ko` |
| `CLAUDE_DIARY_DIR` | 저장 경로 / Diary path | `~/working-diary` |
| `CLAUDE_DIARY_TZ_OFFSET` | UTC 오프셋 / Timezone offset | `9` (KST) |

```bash
# ~/.bashrc, ~/.zshrc, 또는 Windows 환경변수에 추가
export CLAUDE_DIARY_LANG="en"           # English로 변경
export CLAUDE_DIARY_DIR="$HOME/my-diary" # 경로 변경
export CLAUDE_DIARY_TZ_OFFSET="9"        # KST (UTC+9)
```

**Windows 환경변수 설정:**
```powershell
# PowerShell (영구 설정)
[Environment]::SetEnvironmentVariable("CLAUDE_DIARY_LANG", "en", "User")
[Environment]::SetEnvironmentVariable("CLAUDE_DIARY_DIR", "$env:USERPROFILE\my-diary", "User")
```

## 주간 요약 생성 / Weekly Summary

```bash
# 이번 주 요약 / This week
python ~/.claude/hooks/weekly-summary.py

# 특정 주 / Specific week
python ~/.claude/hooks/weekly-summary.py 2026-03-10
```

## Hook 설정 확인

```json
{
  "hooks": {
    "Stop": [
      {
        "hooks": [
          {
            "type": "command",
            "command": "python ~/.claude/hooks/working-diary.py"
          }
        ]
      }
    ]
  }
}
```

## 제거 / Uninstall

```bash
./install.sh --uninstall
```

작업일지 데이터(`~/working-diary/`)는 보존됩니다.

## 요구사항 / Requirements

- Python 3.6+ (`python3` or `python`)
- Claude Code (hooks 지원 버전)

## 팁 / Tips

- **CLAUDE.md에 추가하면 더 좋은 일지가 생성됩니다:**

```markdown
## 작업일지
- 세션 종료 시 작업 내용이 자동 기록됩니다
- 작업 완료/구현/수정 시 한국어로 명확한 요약을 출력해주세요
```

- **Git으로 일지를 관리하면** 작업 이력을 추적할 수 있습니다:

```bash
cd ~/working-diary
git init && git add -A && git commit -m "diary: $(date +%Y-%m-%d)"
```
